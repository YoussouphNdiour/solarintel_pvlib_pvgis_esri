## API Design

- Base path: `/api/v2/`
- Spec: OpenAPI 3.1 (auto-generated by FastAPI)
- Pagination: cursor-based (`?cursor=<opaque>&limit=<int>`)
- Versioning: URL path versioning (`/api/v2/`, `/api/v3/` in future)
- Auth: `Authorization: Bearer <JWT>` header
- Errors: RFC 7807 Problem Details JSON
