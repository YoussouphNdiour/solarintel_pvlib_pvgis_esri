# SolarIntel v2 — Product Vision

## Product Name
SolarIntel v2

## One-Line Description
A SaaS platform for photovoltaic (PV) system sizing, simulation, and monitoring, built for solar installers in West Africa (Senegal).

## Problem Statement
Solar installers in Senegal lack affordable, localised tools for sizing PV installations. Existing tools like PVSyst are expensive and not adapted to local tariff structures (Senelec), weather data, or languages. Installers rely on manual spreadsheets, leading to under- or over-sized systems, reduced ROI for clients, and lost commercial credibility.

## Solution Approach
SolarIntel v2 provides a web + mobile platform that:
- Runs accurate PV simulations using pvlib + PVGIS TMY data anchored to real Senegalese locations
- Offers an AI sizing assistant (LangGraph + Claude API) that recommends system configurations
- Generates professional PDF reports in the style of PVSyst reports
- Integrates Senelec tariff data for accurate economic analysis
- Supports multi-tenant project management for installer firms

## Target User Personas

### Primary: Solar Installer / Engineer
- Works at a small-to-medium solar installation firm in Senegal
- Needs to quote and size PV systems for residential and commercial clients
- Wants credible, professional reports to share with clients and lenders
- Not necessarily a software expert; needs an intuitive interface

### Secondary: Project Manager / Sales Lead
- Manages multiple installation projects across a firm
- Needs a dashboard to track project status, sizing results, and ROI
- Shares PDF reports with clients as part of the sales process

### Tertiary: End Client (indirect)
- Receives the PDF report generated by the installer
- Makes purchasing decisions based on reported ROI and payback period

## Success Metrics
- Time to generate a complete PV sizing report: < 2 minutes
- Report accuracy vs. PVSyst benchmark: within 5% on annual yield
- Installer onboarding time: < 30 minutes from signup to first report
- Monthly active installers: 50 in first 6 months post-launch
- Net Promoter Score (NPS): > 40

## Product Roadmap (High-Level)

### Phase 1 — Foundation (Tracks INFRA-001, AUTH-001)
Infrastructure scaffolding, CI/CD, authentication, user management.

### Phase 2 — Core Simulation (Track SIM-001)
PV simulation engine, ArcGIS map picker, results display.

### Phase 3 — Intelligence (Track AI-001)
AI sizing assistant, LangGraph agent, chat interface.

### Phase 4 — Reporting (Track PDF-001)
Professional PDF report generation, download flow.

### Phase 5 — Dashboard & Economics (Track DASH-001)
Project dashboard, Senelec tariff integration, ROI calculations.
