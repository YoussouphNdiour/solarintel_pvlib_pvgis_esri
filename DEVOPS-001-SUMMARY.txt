MONITORING DASHBOARD (GRAFANA)

Location: http://localhost:3001 (admin/solarintel)
Default: SolarIntel v2 — API & Simulation Dashboard
Refresh: Every 30 seconds
Time window: Last 1 hour

PANELS (6):
1. API Request Rate
   Metric: rate(http_requests_total[1m])
   Unit: requests/second
   Type: Time series with legend table

2. API Latency (P50/P95/P99)
   Metrics: P50, P95, P99 percentiles
   Unit: milliseconds
   Thresholds: yellow@500ms, red@2000ms

3. HTTP 5xx Error Rate
   Metric: rate(http_requests_total{status_code=~"5.."}[1m])
   Unit: errors/second
   Alert: red when > threshold

4. Active WebSocket Connections
   Metric: active_websocket_connections
   Unit: connections
   By: project_id

5. Simulation Runs (by Status)
   Metric: rate(simulation_runs_total[1m])
   By: success/error/cache_hit
   Unit: simulations/second

6. PVGIS Cache Hit Rate
   Metric: (hits / (hits + misses)) * 100
   Unit: percent
   Type: Gauge with thresholds
   Status: red<50%, yellow<80%, green≥80%
